# Claude Guide

## When to use Claude

Use Claude when you want a patient conversational guide for:

- installation walkthroughs
- hardware selection help
- understanding tradeoffs between Lite, Stereo, and Pro
- troubleshooting with plain-language explanations

## Recommended instruction style

Tell Claude:

- your hardware tier
- your operating system
- your current blockers
- that you want one step at a time
- that you want it to stop after each step and wait for confirmation

## Example

“I want to use CHEK EGO Miner with a stereo camera on Windows. Please guide me
step by step, explain what each step is doing, and wait for my result before
you continue. If something fails, help me debug it before moving on.”
