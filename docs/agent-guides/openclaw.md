# OpenClaw Guide

## When to use OpenClaw

Use OpenClaw when you want a fast operator-style agent to:

- drive a concrete install checklist
- validate hardware and app prerequisites
- keep the setup flow moving with short actionable steps

## Recommended instruction style

Tell OpenClaw:

- whether you are `Lite`, `Stereo`, or `Pro`
- your OS
- whether the iOS app is already installed
- whether the camera is already connected
- that you want a strict step-by-step checklist

## Example

“Use a strict checklist. I am setting up CHEK EGO Miner on a Linux laptop with
a stereo camera. Guide me one step at a time, verify each result, and do not
skip any hardware or camera checks.”
