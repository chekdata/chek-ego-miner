# FAQ

## Can I start with just a computer and my own camera?

Yes. Start with the `Lite` tier.

## Do I need an edge machine on day one?

No. The public docs are designed so you can start with `Lite` and grow into
`Stereo` or `Pro`.

## Which agent should I use?

- Use Codex if you want more code-oriented help.
- Use Claude if you want very conversational explanations.
- Use OpenClaw if you want a strict operator-style checklist.

## What if I do not have a stereo camera yet?

Start with `Lite`. Move to `Stereo` later when you need better spatial quality.

## Can I use Windows?

Yes, but hardware and camera behavior still need to be validated carefully on
each setup. Start with the public `doctor` and `readiness` checks.

## Is every host already a one-command install?

Not yet. Check the README and quickstart for the lanes that have already been
verified in this public repo.

## Where do I download contributed datasets?

Use the public dataset portal linked from the README.
