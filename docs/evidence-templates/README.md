# Evidence Templates

These templates keep public validation evidence consistent without leaking
private infrastructure details.

Use them for:

- [First contributor setup](./first-contributor.md)
- [Interrupted install resume](./interrupted-install-resume.md)
- [Agent-guided install](./agent-guided-install.md)
- [Returning user readiness](./returning-user-readiness.md)
- [Camera troubleshooting](./camera-troubleshooting.md)
- [Upload troubleshooting](./upload-troubleshooting.md)
- [Stereo and Pro true-hardware evidence](./stereo-pro-true-hardware.md)

Local diagnostics and `public-e2e` do not upload by default. Upload evidence
belongs only in a separately documented, explicitly requested contribution
flow.
