# Changelog

Initial public foundation for `CHEK EGO Miner`.

- public README in English and Simplified Chinese
- hardware tiers and buying guidance
- agent guides for Codex, Claude, and OpenClaw
- reusable prompts for Lite, Stereo, Pro, and camera troubleshooting
- public-safe diagnostics and probes
- lightweight public CLI:
  - `doctor`
  - `readiness`
  - `quickstart`
  - `charuco`
  - `scan-public-safety`
- privacy, data-license, reward, and FAQ docs
- first public walkthrough records
